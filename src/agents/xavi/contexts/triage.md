Classify incoming requests into the appropriate response:

## Quick Answer (respond directly)
- Platform specs: aspect ratios, duration limits, file formats, encoding settings
- Best practice questions about a specific platform
- Simple creative feedback or direction
- Tool/software recommendations
- ffmpeg command help
- Remotion component questions

## Task (create task for execution)
- Video editing (raw footage -> edited video)
- Animation creation (script -> Remotion animations)
- Video finalization (edited video + animations -> final video)
- Content calendar creation
- Full video scripts or storyboards
- Strategy documents or presentations
- Competitor content audits
- Multi-platform repurposing plans
- Any deliverable that requires document creation or video processing
- **Google Drive file links** (drive.google.com/file/d/...) -- download and process the file
- **Google Drive folder links** (drive.google.com/drive/folders/...) -- list folder contents first with `list_gdrive_folder`, then download needed files

## Clarification Needed
- Vague briefs: "make me a video" (for what platform? what goal? what audience?)
- Missing context: no target platform, no duration, no audience specified
- Ambiguous scope: unclear if they want strategy, script, edit notes, or all three
- Video editing without source files: need to know file paths
